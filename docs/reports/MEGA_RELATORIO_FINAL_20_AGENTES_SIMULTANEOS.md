## 🔴 TOP 20 PROBLEMAS CRÍTICOS (PRIORIDADE ABSOLUTA)

### P0 - CRÍTICO (Resolver ESTA SEMANA)

#### 1. **SEO Completamente Ausente** 🔴🔴🔴

- **Score:** 52/100
- **Impacto:** 90% de perda de tráfego orgânico
- **Gap:** 0 properties indexadas no Google
- **Fix:** Sprint 1 (27h) - Sitemap, Schema.org, Meta tags
- **ROI:** R$ 150K/ano em leads orgânicos

#### 2. **Imagens Sem Otimização** 🔴🔴🔴

- **Problema:** 40-50MB carregados, LCP ~18s
- **Impacto:** Usuários abandonam antes de carregar
- **Fix:** srcset + backend resize (8h)
- **Ganho:** LCP 18s → 3.5s (81% melhoria)

#### 3. **Testing Coverage Baixíssimo** 🔴🔴

- **Coverage:** 15% (precisa 80%)
- **Impacto:** 28 testes falhando (33% flaky)
- **Fix:** 280 horas (~7 semanas)
- **Ganho:** Previne bugs de produção

#### 4. **Drag & Drop Mobile Quebrado** 🔴🔴

- **Problema:** HTML5 DnD sem touch support
- **Impacto:** Kanban não funciona em 40% dos usuários
- **Fix:** Migrar para @dnd-kit (4h)
- **Ganho:** Funcionalidade completa

#### 5. **Monitoring Insuficiente** 🔴🔴

- **Score:** 3.2/5 (Intermediário)
- **Gaps:** Sem tracing, sem métricas, sem alerting
- **Fix:** OpenTelemetry + Prometheus (40h)
- **Ganho:** MTTD 30min → 5min (80% redução)

#### 6. **Backend Dashboard Stats - N+1 Queries** 🔴🔴

- **Problema:** 2-5 segundos de load time
- **Fix:** Query única com COUNT aggregates (3h)
- **Ganho:** 20-50x mais rápido (50-100ms)

#### 7. **Circuit Breaker Ausente** 🔴

- **Problema:** APIs externas sem proteção
- **Risco:** Cascading failures
- **Fix:** opossum library (3 dias)
- **Ganho:** Zero downtime em falhas externas

#### 8. **Recharts Bundle Pesado** 🔴

- **Problema:** +276KB no bundle inicial
- **Impacto:** FCP +800ms
- **Fix:** Lazy loading (4h)
- **Ganho:** -61% bundle, -53% FCP

#### 9. **OAuth Tokens em Plaintext** 🔴

- **Problema:** Tokens sem encryption no DB
- **Risco:** LGPD/GDPR violation
- **Fix:** AES-256-GCM encryption (2h)
- **Ganho:** Compliance total

#### 10. **Health Checks Incompletos** 🔴

- **Problema:** Apenas health.json estático
- **Impacto:** Deploys de versões quebradas
- **Fix:** /api/health completo (2h)
- **Ganho:** Zero-downtime deploys

### P1 - ALTO (Resolver EM 2 SEMANAS)

#### 11. **API Sem Versionamento** 🟡

- **Score:** 0/100
- **Problema:** Breaking changes afetam todos os clients
- **Fix:** /api/v1 versioning (2 semanas)

#### 12. **Virtualização Faltando** 🟡

- **Módulos:** Leads (150+ cards), Rentals, Financial
- **Impacto:** FPS 30-45 (janky)
- **Fix:** @tanstack/react-virtual (16h)
- **Ganho:** 60 FPS smooth

#### 13. **Filtros Sem Debounce** 🟡

- **Problema:** Re-render a cada keystroke
- **Impacto:** 10 caracteres = 1000 operações
- **Fix:** useDebounce(300ms) (30min)

#### 14. **Paginação Inconsistente** 🟡

- **Problema:** Vários padrões de paginação
- **Fix:** Standardizar cursor-based (1 semana)

#### 15. **Logs Sem Estrutura** 🟡

- **Problema:** console.log sem JSON
- **Fix:** Winston structured logging (2h)

#### 16. **Database Sem Constraints** 🟡

- **Problema:** Zero CHECK constraints
- **Risco:** Dados inválidos permitidos
- **Fix:** 4 migrations prontas (1 semana)

#### 17. **Cache Inexistente** 🟡

- **Módulos:** Rentals, Vendas, Settings
- **Impacto:** Refetch completo a cada navegação
- **Fix:** React Query (20h)

#### 18. **Bundle Analysis Missing** 🟡

- **Problema:** Sem enforcement de budgets
- **Fix:** size-limit + CI (1 dia)

#### 19. **Dependency Caching Ausente** 🟡

### Anti-Patterns Encontrados

1. **Fetch Manual ao Invés de React Query**

   ```typescript
   // ❌ BAD
   const [data, setData] = useState();
   useEffect(() => { fetch(...) }, []);

   // ✅ GOOD
   const { data } = useQuery(['key'], fetchFn);
   ```

2. **Componentes Sem Memoização**

   ```typescript
   // ❌ BAD - Re-render a cada mudança pai
   function MyComponent() { ... }

   // ✅ GOOD
   const MyComponent = memo(function MyComponent() { ... });
   ```

3. **Cálculos Pesados Sem useMemo**

   ```typescript
   // ❌ BAD - Executa a cada render
   const filtered = data.filter(...).map(...).sort(...);

   // ✅ GOOD
   const filtered = useMemo(() =>
     data.filter(...).map(...).sort(...),
     [data]
   );
   ```

4. **N+1 Queries no Backend**

   ```typescript
   // ❌ BAD
   const users = await db.select().from(users);
   for (const user of users) {
     user.posts = await db.select().from(posts).where(...);
   }

   // ✅ GOOD
   const usersWithPosts = await db
     .select()
     .from(users)
     .leftJoin(posts, ...);
   ```

5. **State em Memória (Não Escala)**

   ```typescript
   // ❌ BAD - Perde em restart, não funciona em cluster
   const cache = new Map();

   // ✅ GOOD
   const cache = new Redis();
   ```

export function useDebounce<T>(value: T, delay = 300): T {
  const [debounced, setDebounced] = useState(value);

  useEffect(() => {
    const timer = setTimeout(() => setDebounced(value), delay);
    return () => clearTimeout(timer);
  }, [value, delay]);

  return debounced;
}
